# rails6-compose

Docker を用いて Elixir/Phoenix アプリケーションの開発・学習を始めるための設定ファイル等のセット

## 構築される環境

* Debian 10 (Buster)
* Erlang/OTP 22.3.4
* Elixir 1.10.4
* Phoenix 1.5.4.
* Node.js 10.22.0
* npm 6.14.6

## 必要なソフトウェア

* Docker 18 以上
* Git 2.7 以上

## 動作確認済みのOS

* macOS 10.14 Mojave
* Ubuntu 16.04
* Ubuntu 18.04
* Ubuntu 20.04
